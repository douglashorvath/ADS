# Análise e Desenvolvimento de Sistemas

Este repositório contém todos os arquivos que eu criei ou baixei ao longo do meu tempo no curso de Análise e Desenvolvimento de Sistemas. Aqui você encontrará uma variedade de materiais, como projetos, exercícios, anotações e recursos úteis relacionados a cada módulo e disciplina do curso.

## Organização do Repositório

O repositório está organizado em pastas, seguindo a estrutura de módulos e disciplinas do curso. Aqui está uma visão geral da organização:

```
├── Módulo 1
│   ├── Disciplina 1
│   ├── Disciplina 2
│   ├── Disciplina 3
│   └── ...
├── Módulo 2
│   ├── Disciplina 1
│   ├── Disciplina 2
│   ├── Disciplina 3
│   └── ...
├── Módulo 3
│   ├── Disciplina 1
│   ├── Disciplina 2
│   ├── Disciplina 3
│   └── ...
└── ...
```

Dentro de cada pasta de módulo, você encontrará as respectivas pastas de cada disciplina. Os arquivos relacionados a cada disciplina serão armazenados dentro de suas respectivas pastas.
